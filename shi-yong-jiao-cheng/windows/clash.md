# Clash for Windows

* `系统要求：Windows XP 及以上（64-bit）`
* `设备要求：Windows PC / Tablet`

{% hint style="info" %}
Clash 不兼容 SSR 协议，且当前最新版本不再支持 chacha20 加密方式
{% endhint %}

{% hint style="warning" %}
Windows 10 第一次运行会提示被阻止，点击「更多信息」 - 「仍要运行」
{% endhint %}

![](../../.gitbook/assets/2018-11-16-15.54.06.jpg)

1. 输入邮箱/密码登陆
2. 勾选 \[System Proxy\] 即可
