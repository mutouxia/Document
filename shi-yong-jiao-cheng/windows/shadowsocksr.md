# ShadowsocksD

* `系统要求：Windows XP 及以上`
* `软件版本：ShadowsocksD`
* `设备要求：Windows PC / Tablet`

{% hint style="info" %}
ShadowsocksD 不兼容 SSR 协议
{% endhint %}

1. 右键单击桌面右下角的托盘图标，点击订阅 - 管理
2. 将在用户中心复制的订阅地址填入“订阅链接”右侧的编辑框，点击\[添加\]
